# Zomato Data Analysis Project

## 📊 Overview
This project analyzes Zomato dataset using Python and Power BI to extract business insights and trends.

## 🔧 Tools Used
- Python (Pandas, NumPy, Matplotlib)
- Excel
- Power BI

## 📌 Key Insights
- Identified top-performing locations based on sales
- Analyzed cost distribution by food type
- Found impact of ratings and pricing on sales

# 📁 Files Included
- clean_zomato.xlsx → Cleaned dataset
- zomato_eda.ipynb → Data analysis in Python
- zomato_powerbi_dashboard.pbix → Interactive dashboard

# 📈 Outcome
Built an interactive dashboard to track performance, sales trends, and customer preferences.
